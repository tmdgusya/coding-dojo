# Rust Dojo - minigrep

Rust를 배우기 위한 수련장입니다. **minigrep** (간단한 파일 검색 도구)를 만들면서 Rust의 핵심 개념을 마스터하세요.

## 빠른 시작

```bash
cd rust-dojo

# 테스트 실행 (현재 모두 실패 상태)
cargo test

# 특정 임무만 테스트
cargo test mission_1    # 소유권
cargo test mission_2    # 에러 처리
cargo test mission_3    # 구조체
cargo test mission_4    # 트레이트
cargo test mission_5    # 생명주기
```

## 학습 경로

| 임무 | 주제 | 핵심 개념 |
|------|------|----------|
| 1 | 소유권 (Ownership) | String vs &str, move semantics |
| 2 | 에러 처리 | Result, Option, ? 연산자 |
| 3 | 구조체 | struct, impl, 메서드 |
| 4 | 트레이트 | trait, 제네릭, derive |
| 5 | 생명주기 | 'a, 참조의 유효 범위 |
